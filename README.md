# poo-projeto-final

## Para rodar o código com sucesso:
Se certifique de ter o Java Development Kit e o Maven instalado em seu computador e a extensão redhat ou similar no VSCode. 
No terminal do VSCode, digite ls e veja se esta dentro do diretório correto. Você deve estar dentro de /sistema-veterinário/ para funcionar.
Caso tudo esteja certo, abra o terminal, digite "mvn compile" e confirme.
A seguir, digite "mvn exec:java -Dexec.mainClass="Main" e confirme.
O código deve se iniciar em seu terminal.
